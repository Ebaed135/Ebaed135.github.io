Things I've Built

★ Senior Design · 2025–2026
ALICE — ACO Intelligence Engine

An AI-powered MCP (Model Context Protocol) server built for Pathways Health Partners, a healthcare sponsor managing Accountable Care Organizations. ALICE routes natural language clinical queries through a RAG pipeline, using sentence-transformer embeddings and FAISS similarity search to match queries against pre-validated SQL templates — producing accurate, auditable results without freeform query generation.

Python MCP Server FAISS sentence-transformers RAG PostgreSQL Claude API
